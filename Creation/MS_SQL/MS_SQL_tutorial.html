Как установить MS_SQL

  1. установить дома SQL Server Express 2008 и SQL Server Management Studio Express (2008-ой хватит с головой, однако, можно поставить любую другую экспресс версию 2012 или 2014).
    для пущей надежности на Win 7 (и старше) можно это делать от имени администратора "Run as administrator..."
  2. создать базу QALight:
    • открыть Studio и подключиться на установленный сервер SQLExpress
    • открыть файл QAL_DB_Creation.sql и кликнуть Execute, либо нажать F5
  3. наполнить БД данными:
    уже в открытой Studio загрузить и выполнить файл QAL_Data.sql, предварительно убедиться, что активна база QALight (команда use или выпадающий список рядом с кнопкой Execute)
  4. Проверить сщдержимое базы. "Поиграться" и функциональностью самой студии
  5. построить диаграмму базы
