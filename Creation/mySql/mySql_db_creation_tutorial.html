How to setup mySql Db

  1. To enter in mysql type in terminal:

    mysql -uroot -h localhost

  2. After this you should be inside mysql. Try to enter a commnad:

    show databases;

    If some results appeared, all is fine.

  3. After entering mysql, execute 2 commands:

    source [..your path to file../MySQL-QAL-Creation.sql

    source [..your path to file..]/MySQL-QAL-Data.sql

    If no errors happened, databases has been created.

  4. In order to check, enter a command:

    show databases;

    You shoud see database "qalight".
